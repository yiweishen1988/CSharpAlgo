There are one hundred ids from 1 to 100. There are n collection of variable. 
Each collection contains numers between 1 to 100. 
count the total number of arrangements to make each collection contains a 
unique number.

Constraints: 1 <= n <= 10 Example:

For example: 
n=3
5 100 1     // First Collection
2           // Second Collection
5 100       // Third Collection

Output:
4

Explanation: All valid possible ways are (5, 2, 100),  (100, 2, 5), (1, 2, 5) and  (1, 2, 100)
